모든 한계를 부수다

🛡️

API 키 철통 보안

가장 걱정하셨던 보안 문제, 완벽히 해결했습니다. 입력하신 API 키는 오직 사용자 본인의 웹 브라우저(Local Storage)에만 암호화되어 저장됩니다. 서버리스 아키텍처를 통해 NovelAI 본사로 직행하며, 외부 서버로의 유출은 0%입니다.

📜

자동 기록 및 원클릭 복구

생성된 모든 이미지와 상세 설정(시드, 스텝, 샘플러 등)이 브라우저에 자동 기록됩니다. 과거의 빛나는 작품을 클릭 한 번으로 불러오고(Load), 모든 프롬프트를 통째로 복사할 수 있는 강력한 히스토리 모달을 제공합니다.

💎

V4.5 & 다중 캐릭터 완벽 지원

최신 NAI V4.5 모델은 물론, 강력한 Character Prompt(캐릭터별 프롬프트 분리) 기능을 네이티브로 지원합니다. 1boy, 1girl 등 여러 캐릭터가 섞이지 않도록 정교한 컨트롤이 가능합니다.

🪙

실시간 Anlas 비용 계산

구독 중인 티어 정보를 자동으로 불러와, 해상도와 스텝 수에 따른 소모 Anlas 비용을 생성 전에 실시간으로 계산해 줍니다. 3티어(Opus) 유저의 무료 생성 구간(1024x1024 이하)도 정확히 인식합니다.

🎨

전문가용 고급 설정 (Advanced)

공식 사이트의 모든 고급 기능을 담았습니다. Karras, Exponential 등의 노이즈 스케줄, 다양한 샘플러(Euler Ancestral, DPM++ 등), 프롬프트 가중치 리스케일, 그리고 고품질 태그 자동 추가 기능까지 빠짐없이 지원합니다.

✨

편안한 UI/UX 경험

눈이 편안한 다크/라이트 테마 전환, 프롬프트 가중치에 따라 색상이 변하는 실시간 하이라이팅, 라이트박스 이미지 뷰어 등 그림 생성에만 집중할 수 있는 쾌적한 인터페이스를 자랑합니다.
